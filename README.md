# Manual QA SauceDemo E-Commerce Application

This repository contains complete QA documentation for a sample e-commerce application.

## Included Documents
- Test Plan
- TestCase Document
- Functional Test Cases
- Requirement Traceability Matrix (RTM)
- Sample Defect Report
- Test Summary Report

## Scope Covered
- Login functionality
- Product listing
- Add to cart
- Checkout workflow
- Logout

## Testing Types
- Functional Testing
- Regression Testing
- Negative Testing
- UI Validation

This project demonstrates end-to-end manual QA process and documentation standards.
